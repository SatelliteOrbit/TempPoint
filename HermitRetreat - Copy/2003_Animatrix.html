The Animatrix (2003)

DVD

Rating:

PG+

Language(s):

English

Synopsis:

An anthology containing 9 animated short films set in the Matrix universe.

Personal Comments:

I've only watched a few of the shorts. They are very well-animated and each one is visually-distinct. The stories are usually mediocre, the plots only serve the function of showing some really good animation. Despite that there are some interesting concepts and ideas being presented.
